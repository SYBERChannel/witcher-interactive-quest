# Witcher: Path of Geralt

Interactive RPG story web application.

## Stack

Frontend:
- React
- Vite
- Zustand
- Axios

Backend:
- Node.js
- Express
- PostgreSQL

## Features

- branching story
- inventory system
- random events
- battle system
- leaderboard

## Run project

Start database:

docker compose up -d

Start backend:

cd server
npm install
npm run dev

Start frontend:

cd client
npm install
npm run dev
